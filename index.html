Rules

1. Starting the Game : Press the start button to begin the game. The first level will start.

2. Watch the Pattern : A colored button will flash, indicating the sequence. Remember this color pattern as it grows each round.

3. Repeat the Pattern : Once the game flashes a sequence, it’s your turn to click the buttons in the same order.

4. Level Up: If you correctly match the entire sequence, you’ll level up. The game will then add one more color to the sequence.

5. Game Over : If you make a mistake, the game will display "Game Over" along with your score. To play again, press the start button.

6. Objective : Try to complete as many levels as you can by memorizing and repeating the sequence!
